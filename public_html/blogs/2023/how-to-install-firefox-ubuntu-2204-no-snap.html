How to: Install Firefox in Ubuntu 22.04 Jammy Jellyfish (non-snap variant)
Firefox as a Snap

So since Ubuntu version 22.04, Canonical is defaulting to using the snap version of firefox on a new install. This is bad for many reasons, I wrote a seperate article about it, you can read it here:

Rant on Snaps and open letter to Canonical Linux

In the case of the firefox snap package it’s bad for the following reasons. 1. The snap is very slow when compared to the .deb or build variants of the software, on average from cold boot to start like 30-40% slower. 2. It breaks native GNOME system theming, which also defies the point of the entire GNOME design philosophy. 3. Some core functionalities, if unlucky, don’t work well, such as being able to only save downloaded files in tmp directory. 4. It breaks the normal installation of the gnome-shell which allows you to run GNOME extensions.

Because we are not animals, we are not animale, we are going to replace the snap for a version that works properly. First we will remove the firefox snap. You can install another browser such as chrome as a backup. So first on the commandline do a: "sudo snap remove firefox"

Then we will install wget, if not installed yet: "sudo apt install wget"

Then we will install the following required and recommended packages: "sudo apt install libgtk-3-dev libglib2.0-0 libglib2.0-dev gir1.2-pango-1.0 libstdc++6 libdata-hal-perl"

Go to Firefox websites download page for Linux advanced users on: https://www.mozilla.org/en-US/firefox/linux/ and click on the ‘Download Now’ button.

In the terminal change directory to Downloads: "cd ~/Downloads"

Unpack the archive with: "tar xjf firefox-*.tar.bz2"

Now change to the root useraccount on the system: "sudo su". All next lines will be as root until we exit the account.

Now let's move the firefox files directory to the /opt directory with "mv firefox /opt"

Symlink the executable: "ln -s /opt/firefox/firefox /usr/local/bin/firefox"

download a copy of the Firefox desktop icon file and place it in /usr/local/share/applications with:

"wget https://raw.githubusercontent.com/mozilla/sumo-kb/main/install-firefox-linux/firefox.desktop -P /usr/local/share/applications"

Now exit the root user account with command: "exit". You should have gone back to your regular user.

To confirm successful install open the application with the new icon that you can now find in ‘Applications’ or by searching Firefox in the search. Type 'about:support' in the URL-bar in Firefox. Confirm there are no errors or warnings and that the Application Binary value equals "/opt/firefox/firefox-bin".

Now go and enjoy your fast new version of Firefox.
