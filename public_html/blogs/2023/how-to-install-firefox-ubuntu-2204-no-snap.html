How to: Install Firefox in Ubuntu 22.04 Jammy Jellyfish (non-snap variant)
Firefox as a Snap

So since Ubuntu version 22.04, Canonical is defaulting to using the snap version of Firefox on a new install. This is bad for many reasons, I wrote a seperate article about it, you can read it here.

Because we are not animals, we are not animale, we are going to replace the snap for a version that works better. First we will install Google Chrome before we remove the Firefox snap. You can delete Chrome afterwards if you want to.

For Google Chrome we can find the deb file here.

Then install the downloaded file with the following command:

sudo dpkg -i ~/Downloads/google-chrome-stable_current_amd64.deb

Then we will install wget, if not installed yet use: sudo apt install wget

Then we will install the following required and recommended packages with the following command:

sudo apt install libgtk-3-dev libglib2.0-0 libglib2.0-dev gir1.2-pango-1.0 libstdc++6 libdata-hal-perl

Visit the Firefox page for advanced users here and click on the Download For Linux 64bit button.

After the download inside your terminal change directory to the Downloads folder:

cd ~/Downloads

Unpack the archive with:

tar xjf firefox-*.tar.bz2

Now change to the root user on the system using sudo su. All the following commands will be as the root user until we exit the account.

Let's move the Firefox files directory to the /opt directory with:

mv firefox /opt

Symlink the executable:

ln -s /opt/firefox/firefox /usr/local/bin/firefox

Now we will download a copy of the Firefox desktop icon file, and place it in /usr/local/share/applications with:

wget https://raw.githubusercontent.com/mozilla/sumo-kb/main/install-firefox-linux/firefox.desktop -P /usr/local/share/applications

Now exit the root user account with command: exit. You should have gone back to your regular user now.

To confirm successful installation, open the application with the new icon that we can now find under Applications, or by searching Firefox in the searchfield. After opening Firefox type 'about:support' in the URL-bar. Confirm there are no errors or warnings and that the Application Binary value equals /opt/firefox/firefox-bin.

Now go and enjoy your faster version of Firefox.
