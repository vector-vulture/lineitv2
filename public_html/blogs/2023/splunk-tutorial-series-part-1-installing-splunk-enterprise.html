Splunk Tutorial Series - Part 1: Installing Splunk Enterprise 9.1.0 (On Ubuntu Server 22.04 LTS)

In this first video in a multi-part video series about Splunk, we will be installing Splunk Enterprise on Ubuntu server 22.04 LTS edition.

We will go over some of Splunk's minimum requirements, the installation itself, and configuring the permissions from the root user over to the splunk user after the installation. After that, we will troubleshoot any possible KVstore issues that could arise from switching over the permissions of Splunk Enterprise.

I had to go back and rectify a typo in my admin account by reinstalling Splunk, so that already fixed the KVstore problem for me, but I still included the steps to fix it regardless.
