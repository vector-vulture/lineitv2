How to: Completely remove Snapd from Ubuntu 22.04
Snapd

So since Ubuntu version 22.04, Canonical is defaulting to using the snap version of Firefox on a new install. This is bad for many reasons, I wrote a separate article about that, you can read about it here:

Click here to read more on why snapd is bad and it should feel bad!

Because I’m sick of snaps and them being installed to my system I tend to remove it all entirely, in this guide we will be going over how to do that.

I will show you how to remove all snap packages from your system, as well as removing the snap software store itself, all the daemon services and finally purging the .deb package for snapd off of our system.

Warning that this will obviously 1.) stop all snaps you have installed from working, and 2.) might interfere with the correct functionality of the canonical-livepatch (this is not enabled by default and only for paid subscription support users) since this is only available in a snap package. You should always be able to find a non-snap version of all other software, usually the .deb works best.

Uninstalling all Snaps

First lets kick off a “sudo snap list” command to see all the snaps we currently have installed.

Now lets stop all the snapd services with the oneliner: “sudo systemctl disable snapd.service && sudo systemctl disable snapd.socket && sudo systemctl disable snapd.seeded.service"

Now we will remove each snap, one by one. You have to remove them all in a certain order, because some snaps, such as for example core18, depend on other snaps. Just remove them where possible until eventually no more snaps are installed in the list. Don't worry about losing core functionalities of your system. It should not break anything except for your web browser if you use the Firefox snap. Install an alternative browser for now, like Google Chrome, if that's the case.

Then, remove any leftover cached snap data: “sudo rm -rf /var/cache/snapd/”

Now purge/autoremove the actual snapd dpkg package using the following command: “sudo apt autoremove –purge snapd”

Because this action does not clear our home directory automatically, to clear it we have to perform a: “rm -rf ~/snap”
